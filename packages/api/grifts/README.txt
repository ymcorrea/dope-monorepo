# Grift Scripts

This folder contains Go scripts that can be run as one-off commands using the tool "grift".

More information about Grift can be found on [the Grift Github page](https://github.com/markbates/grift).

## TLDR

```sh
  # install grift
  go install github.com/markbates/grift@latest

  # show all available scripts
  grift list

  # run a script
  grift jim
```
